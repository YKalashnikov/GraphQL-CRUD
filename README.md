# GraphQL-CRUD
Create server GraphQL CRUD


Node.js/Express CRUD backend using GraphQL and JSON-Server

## Usage

-Install Dependencies

```bash
$ npm install
```

-Run JSON-Server (Port 3000)

```bash
$ npm run json:server
```

-Run Server (Port 3000)

```bash
$ npm run dev:server
```

-Visit Graphiql IDE

Go to http://localhost:3000/graphql
